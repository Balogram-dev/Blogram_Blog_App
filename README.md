# 📝 Blogram Blog App

**Blogram** is a full-stack AI-powered blogging platform where users can **create, edit, AI-generate, and publish blogs** with **rich text formatting**.  
It’s built to make blogging effortless, creative, and engaging.

---

## 🚀 Features
- ✍️ Create, edit, and publish blog posts  
- 🤖 AI assistance to **generate or edit blog content**  
- 🖋️ Rich Text Editor for formatting (bold, italic, headings, lists, links, etc.)  
- 🖼️ Upload blog thumbnails (with ImageKit integration)  
- 🔑 Admin authentication & login system  
- 📂 Blog categories and filtering  
- 📱 Responsive design for web and mobile  

---

## 🛠️ Tech Stack
**Frontend**  
- React.js  
- Tailwind CSS  
- React Router  
- Context API  

**Backend**  
- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- JWT Authentication  
- ImageKit (for image uploads)  

**Other Tools**  
- Nodemon  
- dotenv  
- CORS
